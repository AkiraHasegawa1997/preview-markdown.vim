# preview-markdown.vim
This is vim plugin that can preview markdown in Vim terminal.

![](https://i.imgur.com/ME5HBWP.png)

## Requirements
- [MichaelMure/mdr](https://github.com/MichaelMure/mdr)

## Instllation
e.g dein.vim

```toml
[[plugin]]
repo = 'skanehira/preview-markdown.vim'
```

## Usage
```vim
:PreviewMarkdown
```
